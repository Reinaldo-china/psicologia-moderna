Guia de Deploy

Passo a passo para publicar o sistema com autenticação e banco de dados

📁Arquivos do projeto

🏠 index.html — Página principal com 3 cards de navegação
🔐 login.html — Tela de login e registro com Supabase Auth
🔑 auth.js — Módulo de autenticação e operações no banco
📊 gantt_psicologia_moderna.html — Gráfico de Gantt interativo
🌳 mapa_relacoes.html — Mapa de relações entre pensadores
📖 area_estudo.html — Caderno de estudo com IA (requer login)
🗄️ supabase_schema.sql — Script SQL para criar as tabelas

1Criar projeto no Supabase

  1. Acesse supabase.com e crie uma conta gratuita
  2. Clique em New Project
  3. Escolha um nome (ex: psicologia-moderna)
  4. Defina uma senha para o banco de dados
  5. Selecione a região South America (São Paulo)
  6. Aguarde a criação (~2 minutos)

2Criar as tabelas no banco

  1. No dashboard do Supabase, vá em SQL Editor (menu lateral)
  2. Clique em New Query
  3. Cole todo o conteúdo do arquivo supabase_schema.sql
  4. Clique em Run
  5. Verifique que apareceu "Success" — 3 tabelas criadas com RLS
💡 O que o SQL cria: 3 tabelas (study_notes, ai_responses, flashcards) com Row Level Security — cada usuário só acessa seus próprios dados.

3Configurar autenticação

  1. No dashboard, vá em Authentication → Settings → Email
  2. Desative a opção "Confirm email" (para login imediato, sem precisar confirmar email)
  3. Salve as alterações

4Copiar URL e chave da API

  1. No dashboard, vá em Settings → API
  2. Copie a Project URL (algo como https://xxxxx.supabase.co)
  3. Copie a anon public key (começa com eyJhbGci...)
✅ A anon key é segura para usar no frontend porque o RLS (Row Level Security) protege os dados — cada usuário só vê os seus.

5Publicar no Netlify

  1. Acesse netlify.com (você já tem conta)
  2. Arraste a pasta com todos os arquivos HTML + JS para o painel do Netlify
  3. Pronto — o site estará online em segundos

Estrutura da pasta para deploy:

psicologia-moderna/
├── index.html
├── login.html
├── auth.js
├── gantt_psicologia_moderna.html
├── mapa_relacoes.html
└── area_estudo.html

(O arquivo supabase_schema.sql não precisa ir para o Netlify — ele é usado apenas no Supabase.)

6Primeiro acesso

  1. Abra o site publicado no Netlify
  2. Vá para a Área de Estudo → será redirecionado ao Login
  3. Na tela de login, clique em ⚙ Configurar Supabase
  4. Cole a URL e a anon key que copiou no passo 4
  5. Clique Salvar configuração
  6. Crie uma conta (aba "Criar Conta") e faça login
💡 A configuração do Supabase é salva no navegador. Cada usuário precisa configurar uma vez no primeiro acesso. Se quiser evitar isso, você pode fixar a URL e key diretamente no código do login.html e auth.js.

⚡Opcional: fixar credenciais no código

Para que os usuários não precisem configurar manualmente, edite o arquivo login.html e substitua as linhas:

let SUPABASE_URL = localStorage.getItem('sb_url') || '';
let SUPABASE_KEY = localStorage.getItem('sb_key') || '';

Por:

let SUPABASE_URL = 'https://SEU-PROJETO.supabase.co';
let SUPABASE_KEY = 'eyJhbGci...SUA-CHAVE-AQUI';

Faça o mesmo no auth.js, na função getConfig():

getConfig() {
  return {
    url: localStorage.getItem('sb_url') || 'https://SEU-PROJETO.supabase.co',
    key: localStorage.getItem('sb_key') || 'eyJhbGci...SUA-CHAVE-AQUI'
  };
},
